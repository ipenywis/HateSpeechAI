# All Datasets must follow this format

## Common Format

| tweet                      | class |
| -------------------------- | ----- |
| hate speech tweet          | 0     |
| offensive tweet (optional) | 1     |
| neutral tweet (optional)   | 2     |
| abusive (optional)         | 3     |
| spam (optional)            | 4     |
| sexism (optional) (N/A)    | 5     |
| insult (optional) (N/A)    | 6     |

    NOTE: Other fields are allowed, but not required.
